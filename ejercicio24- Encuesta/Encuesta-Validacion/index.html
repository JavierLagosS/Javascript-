Ejercicio 24

En esta pregunta se solicita preparar un sitio que simule una encuesta. Para ello, se pide que agregue al menos 5 preguntas estáticas, cada una con una sola alternativa correcta. El obejtivo es que un usuario pueda contestar la encuesta en un máximo de tres ocasiones, e indique el puntaje obtenido (cada pregunta correcta asigna un punto) a través de un mensaje de alerta. En caso que obtenga cinco puntos se debe indicar el puntaje informando que se han contesta do correctamente las preguntas, y se debe deshabilitar el botón. En caso que supere la cantidad máxima de intentos posibles, se debe informar a través de un mensaje de alerta y deshabilitar el botón.

Se solicita validar que el usuario haya contestado todas las preguntas antes de procesarlas, lo cual se debe hacer a través de una función adecuada. Además se pide que los elementos asociados a la encuesta estén dentro de un formulario HTML (etiqueta form).

Utilice una o más funciones Javascript para lograr el objetivo, las que deben estar en un archivo con extensión js aparte
